VAW (Vagrant Ansible WordPress) Documentation

Version 0.5.4

Ansible playbooks for website developer, webmaster and WordPress theme/plugin developer.

Requirements

  • Oracle VM VirtualBox >= 5.0
  • Vagrant >= 1.8.4
  • Ansible >= 2.1.0.0

Vagrant plugin (optional)

  • vagrant-hostsupdater
  • vagrant-vbguest
  • vagrant-serverspec
